# Прикладной искусственный интеллект в системах умного дома

**Курс:** 3–4 курс бакалавриата / 1 курс магистратуры  
**Трудоёмкость:** 5 ЗЕ (180 часов)  
**Форма аттестации:** Экзамен + защита проекта  
**Язык:** Русский

## 🎯 Цель дисциплины

Обучающийся должен быть способен в составе **междисциплинарной команды** самостоятельно:
- спроектировать интеллектуальную функцию умного дома;
- реализовать прототип;
- проверить и интегрировать с исполнительными устройствами;
- оценить качество и обеспечить безопасный контроль человеком.

## 📚 Целевая аудитория

- Программисты
- Электронщики
- Специалисты по автоматике
- Проектировщики и архитекторы
- Обучающиеся инженерных направлений

## 🎓 Образовательный результат

**Прототип интеллектуальной функции умного дома**, реализующий полный цикл:

\[
\text{Сенсор} \to \text{Данные} \to \text{Модель ИИ} \to \text{Решение} \to \text{Физическое/Симулированное действие}
\]

## 🏆 Целевые компетенции КРМ

- **ML-2, ML-4:** Машинное обучение
- **DL-1, DL-3, DL-4:** Глубокое обучение
- **ML-6:** Применение ИИ в приложениях
- **LC-5:** Жизненный цикл ИИ-систем
- **AI S-1:** Безопасность и надёжность ИИ

## 👥 Роли в проекте

1. **ML Engineer (Applied)** — разработка и адаптация моделей
2. **AI Architect (IoT/Edge)** — архитектура системы
3. **Data Engineer (IoT)** — управление сенсорными данными
