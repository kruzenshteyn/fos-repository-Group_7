▎Глоссарий по компьютерному зрению и глубокому обучению

▎A

Accuracy (Точность классификации) — доля правильно классифицированных объектов относительно общего количества объектов.

Activation Function (Функция активации) — нелинейная функция, применяемая к выходу нейрона для формирования способности сети моделировать сложные зависимости.

Adam (Adaptive Moment Estimation) — популярный алгоритм оптимизации, использующий адаптивную настройку скорости обучения.

Adaptive Thresholding — метод бинаризации изображения с вычислением порога отдельно для различных областей изображения.

Anchor Box — заранее заданный ограничивающий прямоугольник определённого размера и пропорций, используемый при обучении некоторых моделей детекции объектов.

Annotation (Аннотация) — описание объектов на изображении, используемое при обучении моделей.

API (Application Programming Interface) — программный интерфейс взаимодействия различных приложений.

ArcFace — современная модель распознавания лиц, использующая угловую функцию потерь.

Artificial Intelligence (AI) — область информатики, занимающаяся разработкой интеллектуальных программных систем.

Augmentation (Аугментация данных) — искусственное увеличение объема обучающего набора данных путем преобразования изображений.

▎B

Backbone — базовая нейронная сеть, извлекающая признаки изображения.

Background Subtraction — метод обнаружения движущихся объектов путем вычитания фона.

Batch — группа обучающих примеров, одновременно подаваемых на вход нейронной сети.

Batch Normalization — метод нормализации промежуточных данных нейронной сети.

Benchmark — стандартный набор данных или тестов для сравнения алгоритмов.

Binary Classification — классификация по двум классам.

Biometric Identification — идентификация личности по биометрическим признакам.

BlazeFace — компактная модель обнаружения лиц компании Google.

BlazePose — модель оценки позы человека для мобильных устройств.

Bounding Box — ограничивающий прямоугольник, определяющий положение объекта на изображении.

▎C

Class — категория объекта, определяемая моделью.

Classification — задача определения принадлежности объекта к определённому классу.

CNN (Convolutional Neural Network) — сверточная нейронная сеть.

COCO Dataset — один из наиболее известных наборов данных для обучения моделей компьютерного зрения.

Confidence Score — вероятность правильности распознавания объекта.

Confusion Matrix — таблица ошибок классификации.

Convolution — операция свертки.

Cross Entropy Loss — функция потерь для задач классификации.

CUDA — программная платформа NVIDIA для вычислений на GPU.

CVAT — инструмент разметки изображений и видеоданных.

▎D

Data Augmentation — увеличение обучающего набора данных.

Dataset — совокупность данных для обучения и тестирования модели.

Deep Learning — глубокое обучение.

Detection — обнаружение объектов.

Dilation — морфологическая операция расширения объектов.

Dropout — метод регуляризации нейронных сетей.

▎E

Edge AI — выполнение моделей искусственного интеллекта непосредственно на периферийных устройствах.

Embedding — компактное векторное представление объекта (например, лица).

Epoch — полный проход по обучающему набору данных.

Erosion — морфологическая операция уменьшения объектов.

Evaluation — оценка качества модели.

▎F

False Negative (FN) — объект существует, но модель его не обнаружила.

False Positive (FP) — модель обнаружила объект там, где его нет.

Face Alignment — выравнивание лица перед распознаванием.

Face Detection — обнаружение лица.

Face Embedding — числовое описание лица.

Face Recognition — распознавание личности по изображению лица.

Feature — информативный признак.

Feature Extraction — извлечение признаков.

Fine-tuning — дообучение предварительно обученной модели.

FLOPS — количество операций с плавающей точкой в секунду.
FPS (Frames Per Second) — количество кадров, обрабатываемых за секунду.

▎G

GAN (Generative Adversarial Network) — генеративно-состязательная сеть.

GPU — графический процессор.

Ground Truth — эталонная разметка данных.

▎H

Haar Cascade — классический алгоритм обнаружения объектов.

Heatmap — карта вероятностей расположения объектов или ключевых точек.

Hidden Layer — скрытый слой нейронной сети.

Histogram Equalization — выравнивание гистограммы изображения.

HOG (Histogram of Oriented Gradients) — дескриптор ориентированных градиентов.

▎I

Image Classification — классификация изображений.

Inference — применение обученной модели для обработки новых данных.

Inference Time — время выполнения одного прохода модели.

IoU (Intersection over Union) — показатель перекрытия двух ограничивающих прямоугольников.

IP Camera — цифровая камера, передающая видеопоток по сети.

▎J

JPEG — формат хранения изображений с потерями.

▎K

Keypoint — ключевая точка человеческого тела.

Knowledge Distillation — перенос знаний от большой модели к компактной.

▎L

Label — метка класса.

Latency — задержка обработки данных.

Learning Rate — скорость обучения.

Loss Function — функция потерь.

LSTM — рекуррентная сеть для анализа последовательностей.

▎M

mAP (Mean Average Precision) — основная метрика качества детекторов объектов.

Mask R-CNN — модель сегментации изображений.

MediaPipe — библиотека Google для анализа человека.

Mini-batch — небольшая группа объектов при обучении.

MobileNet — компактная сверточная сеть.

MobileNet SSD — облегчённая модель обнаружения объектов.

Model Compression — уменьшение размера модели.

MoveNet — компактная модель оценки позы.

MQTT — протокол обмена сообщениями для IoT.

▎N

Neural Network — нейронная сеть.

NMS (Non-Maximum Suppression) — алгоритм удаления повторяющихся ограничивающих прямоугольников.

Normalization — приведение данных к единому диапазону.

NPU (Neural Processing Unit) — специализированный процессор для ускорения выполнения нейронных сетей.

▎O

Object Detection — обнаружение объектов.

Object Tracking — отслеживание объектов.

ONNX — универсальный формат хранения моделей.

OpenCV — библиотека компьютерного зрения.

OpenVINO — пакет оптимизации инференса Intel.

Optimizer — алгоритм оптимизации обучения.

Overfitting — переобучение модели.

▎P

Padding — добавление граничных элементов к изображению.

Pose Estimation — оценка позы человека.

Precision — точность положительных предсказаний.

Preprocessing — предварительная обработка данных.

Pruning — удаление малозначимых параметров модели.

PyTorch — библиотека глубокого обучения.

▎Q

Quantization — уменьшение разрядности представления весов модели.

▎R

Recall — полнота обнаружения.

ReLU — функция активации Rectified Linear Unit.

ResNet — семейство глубоких остаточных сетей.

REST API — программный интерфейс взаимодействия по HTTP.

RGB — цветовая модель изображения.

RNN — рекуррентная нейронная сеть.
